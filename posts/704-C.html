C. Maximum width

題目來源：Codeforces Round 704 (Div. 2)

題目敘述：給兩個字串 s 和 t 其長度分別為 n 和 m 要找到一組 p_1 p_2 p_3 ... p_m (increasing order) 的片列使得 s[p_i]=t[i] 一組 p 序列的寬是 max(p_i+1 - p_i)

先找一組最小的 p 序列 從 s 的開頭開始對 t 紀錄 pos 遇到 t[i] 就記錄下來並繼續 再從 s 的後面進行一樣的紀錄 並對於 pos 的差進行比較


初始：

int n,m;cin>>n>>m;
string s,t;cin>>s>>t;
int pos[n+1];
int cur=0;

紀錄第一個出現 t[i] 的位置

for(int i=0;i<m;i++){
    while(s[cur]!=t[i])cur++;
    pos[i]=cur++;
}

紀錄最後一個出現 t[i] 的位置

cur=n-1;
for(int i=m-1;i>0;i--){
    while(s[cur]!=t[i])cur--;
    ans=max(ans,cur-pos[i-1]);
    cur--;
}
