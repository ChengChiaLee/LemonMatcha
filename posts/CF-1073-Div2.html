pA 水 先填顏色再 sort 可以的話就行 不行就不行 題目只是把要求講得很複雜而已

pB 給一個數列 重新排列使得 MEX(1 2 3 … i) != MEX(i i+1 … n) 把不同 case 列出來就行

- 如果整個數列都沒 0 那前後的 MEX 都會是 0 所以不行

- 如果不只一個 0 那一定要有 1 否則當兩個 0 分開時前後的 MEX 都會是 1 符合就輸出 YES 不符合就 NO

這感覺還需要嚴謹證明 但直覺上沒問題 上傳 AC 好耶

pC 博弈論題 會改一串 binary 要選擇一部分遞減的數列 sort 後放回 選不了就輸了

如果一開始就是 sort 好的 Bob 就贏了 如果沒 sort 好 那 Alice 就可以 sort 感覺 Alice 應該是可以一次就 sort 成功 但要怎麼構造出解才是問題 先做一次 prefix sum 再從後面數 0 如果 cnt0==ps[i] 那這個點以前的 1 都移到 這個點之後的 0 就行 上傳 WA 檢查一下 邊界有問題 修一下 AC

pD1 括號題 要選出一個 sub bracket 使得他有比較早出現的 “(“

研究了一下感覺只要在一組比較深的括號組前有一組比較淺的括號組 移除較淺的括號組就行 上傳 WA (ÒДÓױ)

後來發現移除的未必是要 RBS 只要留下來的對就行 所以只要在這括號的前一半找到 “)” 就行 因為能同時一處後面的 讓剩下的配對起來扔是正常的括號 上傳 AC
